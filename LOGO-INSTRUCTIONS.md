# Logo Setup Instructions

Please add the following logo image files to the project root directory:

## Required Logo Files:

1. **enactus-logo.png** - Enactus logo (will appear on top left)
2. **iit-roorkee-logo.png** - IIT Roorkee logo (will appear on top right, left side)
3. **stc-logo.png** - STC logo (will appear on top right, right side)

## Recommended Image Specifications:

- **Format**: PNG with transparent background
- **Height**: 60-80px (will be auto-scaled to 60px)
- **Aspect Ratio**: Maintain original logo proportions
- **Quality**: High resolution for crisp display

## How to Add Logos:

1. Download or obtain the official logo files
2. Rename them exactly as:
   - `enactus-logo.png`
   - `iit-roorkee-logo.png`
   - `stc-logo.png`
3. Place them in the root directory of this project (same folder as index.html)
4. Refresh your browser to see the logos

## Current Status:

The HTML files are already configured to display these logos. Once you add the actual image files, they will automatically appear in the header of all pages.
